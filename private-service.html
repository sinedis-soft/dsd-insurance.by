Процесс страхования

  • ШАГ 1. ДАННЫЕ

    Заполните форму на сайте по выбранному типу страховки. Информацию достаточно заполнить один раз. В следующий раз основная часть формы будет заполняться автоматически.

  • ШАГ 2. ОБРАБОТКА ДАННЫХ

    Первый освободившийся специалист примет Вашу заявку в обработку. Будьте готовы, что специалист свяжется с Вами для уточнения данных

  • ШАГ 3. ПОДГОТОВКА КОММЕРЧЕСКОГО ПРЕДЛОЖЕНИЯ

    После обработки данных вам будет предоставлено коммерческое предложение и / или счет на оплату.

  • ШАГ 4. ОПЛАТА

    Не всегда данные об оплате приходят быстро. Отправьте данные по оплате специалисту, который Вас обслуживает, для ускорения выписки полиса.

  • ШАГ 5-А. ВЫПИСКА ПОЛИСА

    После получения извещения о оплате, оператор отправит вам ссылку на скачивание электронного полиса, если страховая компания работает с электронными полисами.

  • ШАГ 5-Б. ВЫПИСКА ПОЛИСА

    После получения извещения о оплате, оператор свяжется с Вами по вопросу доставки полиса, если страховая компания работает с бумажными полисами.

  • ШАГ 6-А. ПОЛУЧЕНИЯ ПОЛИСА

    Просто скачайте полис себе на устройство, перейдя по ссылке

  • ШАГ 6-Б. ПОЛУЧЕНИЯ ПОЛИСА

    Если Вы находитесь в Минске, наш курьер доставит Вам полис. В противном случае мы отправим полис по почте.
